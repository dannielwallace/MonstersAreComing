# Monsters are Coming! 复刻开发调研文档

调研日期：2026-04-29  
目标：为制作一款受 Steam 游戏《Monsters are Coming!》及其 `Rock & Road` 宣传副标题启发的同类玩法原型，整理玩法、系统、内容、技术与制作计划。

## 文档索引

- [01_research_brief.md](01_research_brief.md)：游戏身份、核心卖点、公开资料摘要。
- [02_game_design_document.md](02_game_design_document.md)：可执行 GDD，定义目标体验、核心循环、MVP 范围。
- [03_systems_design.md](03_systems_design.md)：资源、建筑、敌人、战斗、波次、地图生成等系统拆解。
- [04_content_reference.md](04_content_reference.md)：内容清单模板、可替代命名、数值原型表。
- [05_technical_plan.md](05_technical_plan.md)：工程架构、数据结构、AI、存档、工具链建议。

- [07_sources_and_ip_notes.md](07_sources_and_ip_notes.md)：资料来源、可信度、版权与规避建议。

## 复刻原则

本项目建议做“机制同类原型”，不要复制原作名称、Logo、素材、地图、角色、音效、UI 布局、文本、关卡配置或专有内容命名。可以学习的是公开可观察的高层玩法结构：移动基地、自动战斗、资源采集、Roguelite 升级、波次防守和建造选择。
