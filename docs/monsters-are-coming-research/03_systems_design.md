# 03. 系统设计

## 移动城镇

### 组成

- 核心：拥有生命、移动速度、建造容量、模块挂点。
- 路线控制器：沿预生成样条或导航路径前进。
- 建造网格：以核心为中心的局部格子，随城镇移动。
- 防守半径：敌人进入后优先攻击城镇或最近防御。

### 参数

| 参数 | MVP 建议 |
| --- | ---: |
| 核心生命 | 1000 |
| 基础移速 | 1.0 单位/秒 |
| 建造半径 | 8 格 |
| 初始挂点 | 12 |
| 每分钟怪物强度增长 | 12%-18% |

## 资源系统

| 资源 | 来源 | 用途 | 设计目的 |
| --- | --- | --- | --- |
| 木材 | 树、废车、营地 | 基础墙、箭塔、采集站 | 早期高频资源 |
| 石料 | 岩石、矿点 | 高级墙、投石器、核心升级 | 中期瓶颈 |
| 金币 | 宝箱、精英、商人 | 特殊建筑、升级刷新 | 决策资源 |
| 经验 | 击杀、事件 | 玩家升级 | Roguelite 成长 |

采集规则建议：

- 靠近资源点自动采集，避免额外按键。
- 大资源点需要持续停留，制造离城风险。
- 资源掉落会被城镇磁吸，但远距离掉落需要玩家拾取。

## 建筑系统

### 建筑类别

- 防御：墙、尖刺、护盾塔。
- 攻击：箭塔、炮塔、投石器、火焰塔。
- 生产：采集站、矿站、税所。
- 支援：修理站、增幅塔、减速塔。
- 召唤：兵营、无人机巢、路障工坊。

### 建造规则

- 建筑必须放在城镇网格内。
- 城镇移动时建筑随核心整体移动。
- 外圈建筑更容易承伤，内圈建筑更安全但射界受限。
- 建筑拥有体积和阻挡，敌人会攻击阻路建筑。

## 战斗系统

### 目标选择

攻击建筑：

1. 优先攻击射程内最近敌人。
2. 若有特殊词条，可改为优先血量最高、最低、最快、正在攻击核心的敌人。
3. 投射物使用简单预判，MVP 可先不做复杂弹道。

敌人：

1. 默认追击城镇核心。
2. 若被玩家攻击，可短暂转向玩家。
3. 攻城敌人优先攻击建筑。
4. 自爆敌人在接近目标后爆炸。

### 伤害类型

| 类型 | 特征 | 适合单位 |
| --- | --- | --- |
| 穿刺 | 单体高频 | 箭塔、游侠 |
| 钝击 | 范围溅射 | 投石器、Boss |
| 火焰 | 持续伤害 | 火塔、炼金 |
| 电击 | 连锁 | 高级塔 |
| 工程 | 对建筑/护甲加成 | 攻城怪、炮塔 |

## 波次系统

### 时间轴原型

| 时间 | 内容 |
| ---: | --- |
| 0:30 | 第一波小怪 |
| 2:00 | 快速怪加入 |
| 4:00 | 精英小队 |
| 6:00 | 远程怪加入 |
| 8:00 | 坦克怪加入 |
| 10:00 | 自爆怪加入 |
| 12:00 | 大规模混合波 |
| 15:00 | Boss |

### 刷怪规则

- 刷在屏幕外或战争迷雾边缘，保持公平预警。
- 以城镇移动方向为主压迫，侧翼偶尔出现骚扰。
- 每波使用预算系统：怪物有点数，随时间提高预算。
- 特殊地形可生成巢穴，玩家摧毁后减少后续压力。

## 升级系统

### 局内升级池

- 玩家升级：移速、采集速度、拾取范围、生命、技能伤害。
- 建筑升级：射速、射程、耐久、溅射、穿透。
- 城镇升级：核心生命、修理效率、建造容量、资源折扣。

### 稀有度

- 普通：小幅线性提升。
- 稀有：改变优先级或附加效果。
- 史诗：形成构筑核心，例如箭塔弹射、城墙反伤、资源点爆裂。

## 地图系统

MVP 使用“主路径 + 周边资源簇”的生成方式：

- 生成一条从起点到终点的平滑路径。
- 沿路径每隔固定距离布置资源簇。
- 远离路径的资源更丰富，但敌人风险更高。
- 关键节点生成事件或精英战。

## UI 系统

- 左上：城镇生命、当前目标、下一波倒计时。
- 顶部：资源条。
- 右侧：迷你波次提示和临时任务。
- 底部：建筑快捷栏。
- 升级界面：三选一，显示影响对象、数值变化、稀有度。

## 结算系统

- 显示行进距离、击杀数、采集量、建造数量、最高伤害建筑、失败原因。
- 奖励局外货币或蓝图碎片。
- 推荐下一局目标：尝试新建筑、提高难度、解锁角色。
